## 统计基础重要知识点
### 古典概型
#### 条件概率
$$P(B|A) = \frac{ P(AB) } {P(A)}$$
#### 全概率公式
$$P(A)=P(A|B_{1}) P(B_{1}) + P(A|B_{2}) P(B_{2}) + ... + P(A|B_{n}) P(B_{n})$$
#### 贝叶斯公式
$$P(B_{i}|A) = \frac{P(A|B_{i}) P(B_{i})} {\sum_{j}^{n} P(A|B_{j})P(B_{j})}$$
### 几何概型
$$p(A) = \frac{构成时间A的区域长度（面积或体积）} {全部结果构成的区域长度（面积或体积）}$$
